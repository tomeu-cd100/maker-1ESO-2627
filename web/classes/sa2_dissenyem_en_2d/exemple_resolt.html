Exemple resolt (model per a l'alumnat) — SA2

Nota docent: projecta'l després del primer disseny i fes que l'alumnat identifiqui quins criteris d'èxit compleix abans d'explicar-ho tu (apropiació dels criteris).

🔑 Exemple de producte: el marcapàgines del Marc

 +------------------------------+
 |   ( o )  <- forat per a la borla
 |   .----.                     |
 |   | ☼  |  <- imatge vectoritzada (un sol) gravada
 |   '----'                     |
 |                              |
 |   "Llegir és viatjar"  <-- missatge gravat (text → camí)
 |                              |
 |   [marc interior gravat]     |
 +------------------------------+
   Mida: 50 x 150 mm · DM 3 mm
   TALL: contorn + forat · GRAVAT: sol + missatge + marc

Per què és un bon producte (🌟): - Conté forma + missatge + imatge vectoritzada (els 3 elements). - El text està convertit a camí (no es desmuntarà a la làser). - Aprofita bé la planxa: el disseny és compacte, deixa poc material malgastat.

📝 Exemple de fitxa (emplenada)

  • Per a qui: per a la meva àvia, que llegeix molt.
  • Missatge: "Llegir és viatjar". Imatge: un sol (vectoritzat d'una icona).
  • Operacions de camí usades: unió per fer el marc; el sol amb diferència.
  • Aprofitament: he posat el marcapàgines en vertical per encabir-ne dos a la planxa.

📔 Exemple d'entrada al diari (ben feta)

Sessió 2: he vectoritzat la imatge del sol (Camí → Vectoritza mapa de bits) i he ajustat el llindar perquè quedés net. El text al principi sortia com a contorn buit; la profe m'ha ensenyat a fer Objecte a camí i a posar-lo de gravat. He comprovat que tot fa 50 mm d'ample. Problema: la imatge tenia massa detalls i el gravat trigava molt → l'he simplificada. Evidència: [captura del fitxer amb les capes].

Per què és bona: mostra una decisió tècnica (simplificar per reduir temps de gravat) i usa el vocabulari clau (vectoritzar, llindar, camí, capa).

⚠️ Contraexemple

  • Deixar el text com a text (no convertit a camí) → pot fallar o canviar de tipografia.
  • Imatge amb massa detall → gravat lentíssim i brut.
  • Disseny escampat per la planxa → es malgasta material.
