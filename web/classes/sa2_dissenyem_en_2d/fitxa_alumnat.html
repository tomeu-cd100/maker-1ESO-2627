Fitxa de l'alumnat — SA2: Marcapàgines a mida

Nom: ___ Equip/parella: _

El repte

Dissenya un marcapàgines (o etiqueta) personalitzat amb tall i gravat. Requisits: - [ ] Forma base (mida orientativa 150 × 50 mm) - [ ] Un missatge o nom gravat (text convertit a camí) - [ ] Una imatge vectoritzada (gravada) - [ ] Un forat per a la borla - [ ] Capes correctes: tall (contorn) i gravat (text + imatge)

Recorda d'Inkscape

  • Operacions de camí: Camí → Unió / Diferència / Intersecció.
  • Text per a làser: escriu el text → selecciona'l → Camí → Objecte a camí.
  • Vectoritzar imatge: importa-la → Camí → Vectoritza mapa de bits → ajusta llindar.
  • Treballa sempre en mil·límetres i comprova la mida final.
